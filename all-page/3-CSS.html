CSS

CSS (Cascading Style Sheets) est un langage utilisé pour décrire la présentation d'une page web en HTML. Il permet de définir l'apparence visuelle des éléments, tels que les couleurs, les tailles, les polices, les marges, et plus encore. Voici une explication simple avec des exemples de sélecteurs et de propriétés.

Une règle CSS est composée de deux parties : Le sélecteur : Il désigne l’élément HTML que vous voulez styliser. Les propriétés : Elles définissent les styles appliqués à cet élément.
